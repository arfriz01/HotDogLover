MVC Profile: Hot Dog Lover

March 03, 2015 by Alicia R. Frizzell

Welcome back! Today we will be talking about MVC and I will discuss a simple profile tutorial that you can try out to see how you like MVC. We will be using Visual Studio 2013 to make our MVC applications. Visual Studio is a great web development tool. It can be used to implement many different platforms such as ASP.NET, MVC and Web Forms.

The profile I have created is one for an avid HotDog fan. I used a combination of models, views and controllers to display and control the data presented to the user. 

"My Favorite Dog on the planet is the Cheddar Cheese Filled Dog at Tony's! 
			You have to go try it out!"

We can allow the user to express their love for dogs with this application.
